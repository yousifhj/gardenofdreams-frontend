## Installation

### Fork and clone repo
```
$ git clone https://github.com/yousifhj/gardenofdreams-frontend
```

### Exnsure backend is running
```
$ http://localhost:3000/
```

### Install dependencies
```
$ npm install
```


### Start Server 
```
$ npm start
```
